# 📺 IPTV Playlist

A curated IPTV playlist built using the **IPTV-org project**, filtered to include only working channels.

---

## 📡 Source

- 🔗 Original Repo: https://github.com/iptv-org/iptv  
- 📂 Playlist Used: https://iptv-org.github.io/iptv/index.m3u
- Nepali: https://iptv-org.github.io/iptv/languages/nep.m3u
- Hindi: https://iptv-org.github.io/iptv/languages/hin.m3u

---

## 🌐 Channels Included

- 🇳🇵 Nepali Channels  
- 🇬🇧 English Channels  

---

## ⚙️ About This Project

This repository is a **filtered collection** of IPTV streams taken from the IPTV-org database.  
Only working and relevant channels are included to keep the playlist clean and fast.

---

## ⚠️ Disclaimer

- ❌ No streams are hosted in this repo  
- 📡 All streams belong to their respective owners  
- 🔄 Channels may go offline or change anytime  
- 📚 This project only organizes publicly available IPTV sources
